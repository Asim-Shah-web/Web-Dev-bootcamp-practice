Full-Stack Web Development Practice (Udemy - Dr. Angela Yu)
===============================

📌 About This Repository:
-------------------------
This repository contains my complete practice work from the "Complete 2024 Web Development Bootcamp" by Dr. Angela Yu on Udemy. It includes a wide range of frontend and backend technologies, with hands-on exercises and projects covering every module in the course.

Throughout this course, I developed a strong interest in web development, and this repo is a reflection of my consistent practice and growing skills across the full stack.

📁 Structure:
-------------
The repository consists of ~38 folders, each corresponding to a particular section/module of the course. These folders include:

✅ HTML & CSS Fundamentals  
✅ Responsive Web Design with Flexbox & Grid  
✅ JavaScript Basics & DOM Manipulation  
✅ Intermediate JS (Functions, Arrays, Objects, Events)  
✅ Node.js & NPM  
✅ Express.js Framework  
✅ Building RESTful APIs  
✅ MongoDB & Mongoose (Database Integration)  
✅ EJS Templating  
✅ Authentication & Security (Intro)  
✅ Basic React (Components, Props, JSX, useState)

🛠️ Technologies Used:
----------------------
- HTML5, CSS3, Flexbox, Grid
- JavaScript (ES6+)
- DOM manipulation
- Node.js (runtime)
- Express.js (server-side framework)
- MongoDB & Mongoose (NoSQL database)
- REST APIs & Routing
- EJS (templating)
- React.js (basic concepts)

🚀 How to Use:
--------------
1. Clone the repository:
   git clone 

2. Browse each folder by topic or module.

3. For Node/Express apps:
   - Run `npm install` to install dependencies.
   - Run `node app.js` or `nodemon app.js` (if installed) to start the server.

4. For React practice:
   - Navigate to the project directory.
   - Run `npm install` and then `npm start`.
